Input Devices

Keyboard

A computer keyboard is an input device that allows a person to enter letters, numbers, and other symbols (together, these are called characters) into a computer. It is one of the most used input devices for computers. Using a keyboard is often called typing. A keyboard contains many mechanical switches or push-buttons called "keys".

Mouse

Many mice have two buttons and a scroll wheel. A computer mouse is an input device that is most often used with a personal computer. Moving a mouse along a flat surface can move the on-screen cursor to different items on the screen. Items can be moved or selected by pressing the mouse buttons (called clicking).
